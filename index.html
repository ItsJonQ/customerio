Fixin' Bootstrap

Bootstrap is great, and it's awesome to see that customer.io is using it. Unfortunately, Bootstrap has a handful of spacing and typography inconsistencies as well as a few performance issues.

That's why I created Lil' B - a bootstrap for Bootstrap!

Check it out on Github

Without Lil' B

Bootstrap's modals are especially bad as far as performance "leaks" go. Check out the browser processes without Lil' B. Notice all the green. That means for every pixel scrolled, the browser has to redraw the entire screen. Yikes.

With Lil' B

Ahh, much better! Like magic, the green is gone. That translates to a much better UX, especially for those folks with older computers.
